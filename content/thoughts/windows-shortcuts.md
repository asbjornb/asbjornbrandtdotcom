# Windows shortcuts

| Windows key + key | action |
| --- | --- |
| R | Run |
| E | File explorer |
| L | Lock |
| D | Desktop |
| Up arrow | Maximize |
| Down arrow | Unlock -> Minimize |
| Left/right arrow | Dock window to left or right half og screen |

Other neat ones:
| Key | Action |
| --- | --- |
| F4 | Inside file explorer goes to path and selects all |
